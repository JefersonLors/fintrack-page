# FinTrack — Landing Page

Landing page oficial do **FinTrack**, um aplicativo móvel desenvolvido como Trabalho de Conclusão de Curso (TCC) no Instituto Federal da Bahia (IFBA), voltado ao gerenciamento inteligente de comprovantes de transações financeiras.

A página tem como objetivo apresentar a solução, seus principais recursos e disponibilizar as versões do aplicativo para download.

---

## 📱 Sobre o FinTrack

O FinTrack é um repositório inteligente de comprovantes que permite registrar, organizar e recuperar notas fiscais, recibos e comprovantes de Pix de forma simples e eficiente.

A aplicação foi projetada com uma premissa central:

> **Seus dados não devem sair do seu dispositivo.**

### Principais características:

- 📷 Captura de comprovantes via câmera
- 📥 Importação direta de outros aplicativos
- 🤖 Extração automática de dados com OCR
- 🔎 Busca textual e semântica (por significado)
- 🧠 Geração de embeddings localmente
- 📊 Relatórios financeiros por categoria e período
- 🔐 Backup criptografado opcional no Google Drive
- 📵 Arquitetura *offline-first*

---

## 🔐 Privacidade

O FinTrack foi desenvolvido com foco em privacidade e autonomia do usuário:

- Nenhum dado é enviado para servidores externos
- Todo o processamento ocorre localmente no dispositivo
- O backup em nuvem é opcional
- Os arquivos são criptografados antes de qualquer envio

## 🚀 Tecnologias

- **Flutter** — desenvolvimento multiplataforma
- **SQLite + Drift** — persistência local
- **Google ML Kit** — OCR on-device
- **Sentence Transformers (MiniLM)** — embeddings semânticos
- **Google Drive API** — backup criptografado opcional
